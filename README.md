# Rainy’s Armor

GUIDE : https://youtu.be/oqhJXBe2Cew
Rainy’s Armor is a progression-style Minecraft mod that adds 6 unique armor sets, powerful weapons, and special abilities that change how you play the game.

Each armor set introduces its own mechanics, effects, and playstyle, from mobility-based movement to high-damage combat abilities and utility tools.

It's made for eductional purpose 

---

# Void Set
![void](https://cdn.hackclub.com/019e9713-7572-748c-b306-eee671872cfc/image.png)

* The Void Armor is crafted using Ender Pearls
* Grants a custom Void effect when the full set is equipped
* Includes the Void Sword with its own abilities
* Part of the Abyss-themed progression system

---

# Bedrock Set
![bedrock](https://cdn.hackclub.com/019e93aa-ad87-78db-baa0-624c73e5bfa1/image.png)

* Bedrock is now craftable using a Bed + Cobblestone placed next to each other
* Unlocks the Bedrock Armor Set
* Includes the Bedrock Sword with a custom ability
* Comes with a unique Bedrock effect tied to the set
* Makes the player way smaller now

---

# Slime Set

* Includes the Slime Armor Set
* Adds the Slime Sling
* The Slime Sling can launch the player into the air for mobility and combat
* One of the most fun and powerful utility items in the mod

---

# Frost Spear
![frost spear](https://cdn.hackclub.com/019e9109-0ee3-7f6f-b596-f855c47baf23/image.png)

* Adds the Frost Spear weapon
* A powerful icy ranged/melee hybrid weapon designed for combat control

---

# TNT Set
![tnt](https://cdn.hackclub.com/019e8c55-5cc8-7048-b586-45476e9a915a/%7BC4405CB1-AB59-463F-B991-DBB556FC57F5%7D.png)

* Adds the TNT Sword
* Includes a custom TNT Sword ability
* Adds crafting recipes for the TNT Sword
* Focused on explosive combat playstyle

---

# Summary

Rainy’s Armor focuses on 6 distinct armor sets with unique mechanics, including movement abilities, combat effects, and custom crafting systems, making progression feel rewarding and diverse.
